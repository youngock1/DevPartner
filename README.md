## 📝 Описание
Этот проект это бот для поиска людей для совместной работы над pet-projetcs и it-sturtup's. В боте реализована tinder system. В режима поиска людей, будут выводиться заполненные ими анкеты пользователю который ищет. И пользователю будет предложено лайкнуть/дизлайкнуть анкету. Также можно будет прекратить просмотр анкет. Когда лайкаешь анкету человека, отправляется уведомление тому человеку с предложением написать человеку который лайкнул. То есть необязательно взаимный лайк для связи с другим человеком. Там уже человек сам решает написать человеку или нет. Дизлайк как бы помечает эту анкету как просмотренную.

## 💻 Использование
Проект размещен в телеграмм боте. Вот инструкция как написать ему:
1. Открываем Telegram
2. Вводим в поиск чатов: `@DevPartner1Bot`
3. Нажимаем в чате на кнопку `/start`
